## üìä SEO Writing Best Practices

### Content Length
- Homepage: 500-800 words
- About: 800-1200 words
- Academics: 1000-1500 words
- Admissions: 800-1200 words
- Other pages: 500-800 words

### Keyword Density
- Aim for 1-2% keyword density
- Use keywords naturally
- Include variations and synonyms
- Don't keyword stuff

### Heading Structure
```html
<h1>Main Page Title (Only ONE per page)</h1>
<h2>Major Section</h2>
<h3>Subsection</h3>
<h4>Minor point</h4>
```

### Internal Linking Best Practices
- Link to 2-5 relevant internal pages from each page
- Use descriptive anchor text
- Don't overdo it
- Link to important pages more often

Example:
```html
<!-- Good -->
<a href="/admissions">Learn about our admission process</a>

<!-- Avoid -->
<a href="/admissions">Click here</a>
```

### Image Optimization
1. **File names**: descriptive, with keywords
   ```
   Good: amazing-angels-classroom-students.jpg
   Bad: IMG_1234.jpg
   ```

2. **Alt text**: descriptive, natural
   ```html
   <img 
     src="classroom.jpg" 
     alt="Students learning in smart classroom at Amazing Angels Public School"
     title="Amazing Angels Smart Classroom"
   />
   ```

3. **File size**: Optimize for web (under 200KB)
4. **Format**: Use WebP when possible, JPG for photos, PNG for graphics
5. **Lazy loading**: Implement for images below the fold

### Call-to-Action (CTA) Optimization
- Make CTAs prominent and action-oriented
- Use contrasting colors
- Place above the fold on key pages
- Examples:
  - "Apply for Admission 2025-26"
  - "Schedule a School Visit"
  - "Download Prospectus"
  - "Call Now: [Phone Number]"
  - "Book a Virtual Tour"
